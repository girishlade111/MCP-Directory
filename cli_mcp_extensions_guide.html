CLI Extensions and MCP Servers Guide

Core / File System
MCP Filesystem MCP
Local file read/write/create access. Most important — Claude/Gemini can directly edit your project files.
npx -y @modelcontextprotocol/server-filesystem /your/project/path
Claude CodeGemini CLI
→ github.com/modelcontextprotocol/servers
MCP Memory MCP
Persistent knowledge graph across sessions. Claude/Gemini remember entities, relations, context between chats.
npx -y @modelcontextprotocol/server-memory
Claude CodeGemini CLI
→ github.com/modelcontextprotocol/servers
MCP Sequential Thinking MCP
Forces structured step-by-step reasoning. Great for complex architecture decisions and debugging.
npx -y @modelcontextprotocol/server-sequential-thinking
Claude CodeGemini CLI
→ github.com/modelcontextprotocol/servers
Dev Tools
MCP GitHub MCP
Create issues, PRs, read repos, list branches — all from within Claude Code or Gemini CLI.
npx -y @modelcontextprotocol/server-github
GITHUB_PERSONAL_ACCESS_TOKEN=your_token
Claude CodeGemini CLI
→ github.com/modelcontextprotocol/servers
MCP Postgres MCP
Query your Postgres/Supabase DB directly. Perfect for LadeStack — schema inspection, data checks inline.
npx -y @modelcontextprotocol/server-postgres postgresql://user:pass@host:5432/db
Claude CodeGemini CLI
→ github.com/modelcontextprotocol/servers
MCP Context7 MCP
Pulls live, version-accurate docs for Next.js, shadcn, Tailwind, etc. No more hallucinated APIs.
npx -y @upstash/context7-mcp@latest
Claude CodeGemini CLI
→ github.com/upstash/context7
MCP Puppeteer MCP
Browser automation — screenshot, click, scrape from CLI. Useful for testing your LadeStack UIs.
npx -y @modelcontextprotocol/server-puppeteer
Claude CodeGemini CLI
→ github.com/modelcontextprotocol/servers
Search & Web
MCP Brave Search MCP
Web search from CLI. Free tier available — get API key from brave.com/search/api.
npx -y @modelcontextprotocol/server-brave-search
BRAVE_API_KEY=your_brave_key
Claude CodeGemini CLI
→ github.com/modelcontextprotocol/servers
MCP Fetch MCP
Fetch any URL, convert to markdown. Read documentation pages, APIs, or competitor sites inline.
npx -y @modelcontextprotocol/server-fetch
Claude CodeGemini CLI
→ github.com/modelcontextprotocol/servers
💡 Qwen CLI note: Qwen CLI (via OpenAI-compatible API) doesn't natively support MCP yet. Use it through Claude Code or Gemini CLI for MCP features. For Qwen model, configure it as custom provider in supported CLI tools via OPENAI_API_BASE override.
Claude Code
EXT Claude Code (VS Code Extension)
Full Claude Code inside code-server/VS Code. Inline chat, file edits, terminal access. Perfect for your DigitalOcean setup.
ext install anthropic.claude-code
code-serverVS Code

EXT Claude Code (JetBrains Plugin)
For IntelliJ/WebStorm users. Same Claude Code features inside JetBrains IDEs.
Settings → Plugins → search "Claude Code"
→ JetBrains Marketplace
VS Code / code-server Extensions (Must Have)
EXT ESLint + Prettier
Auto-format + lint on save. Claude Code output quality improves when code is consistently formatted.
ext install dbaeumer.vscode-eslint esbenp.prettier-vscode
EXT GitLens
See git blame, history, diffs inline. Pairs well with Claude Code's git integration.
ext install eamodio.gitlens
→ VS Code Marketplace
EXT Error Lens
Shows errors/warnings inline on the same line. Claude Code can see these via terminal and fix faster.
ext install usernamehw.errorlens
EXT Tailwind CSS IntelliSense
Autocomplete for Tailwind classes. Essential for your Next.js + shadcn projects.
ext install bradlc.vscode-tailwindcss

Test APIs directly from .http files. Useful alongside your LadeStack API testing tool.
ext install humao.rest-client
Claude Code — claude.json / .mcp.json
Add this to ~/.claude/claude.json or project-level .mcp.json
{ "mcpServers": { "filesystem": { "command": "npx", "args": ["-y","@modelcontextprotocol/server-filesystem","/home/user/projects"] }, "memory": { "command": "npx", "args": ["-y","@modelcontextprotocol/server-memory"] }, "github": { "command": "npx", "args": ["-y","@modelcontextprotocol/server-github"], "env": { "GITHUB_PERSONAL_ACCESS_TOKEN": "YOUR_TOKEN" } }, "postgres": { "command": "npx", "args": ["-y","@modelcontextprotocol/server-postgres","postgresql://user:pass@localhost:5432/ladestack"] }, "context7": { "command": "npx", "args": ["-y","@upstash/context7-mcp@latest"] }, "fetch": { "command": "npx", "args": ["-y","@modelcontextprotocol/server-fetch"] }, "sequential-thinking": { "command": "npx", "args": ["-y","@modelcontextprotocol/server-sequential-thinking"] } } }
Gemini CLI — settings.json
Add to ~/.gemini/settings.json
{ "mcpServers": { "filesystem": { "command": "npx", "args": ["-y","@modelcontextprotocol/server-filesystem","/home/user/projects"] }, "context7": { "command": "npx", "args": ["-y","@upstash/context7-mcp@latest"] }, "github": { "command": "npx", "args": ["-y","@modelcontextprotocol/server-github"], "env": { "GITHUB_PERSONAL_ACCESS_TOKEN": "YOUR_TOKEN" } }, "fetch": { "command": "npx", "args": ["-y","@modelcontextprotocol/server-fetch"] } } }
→ Gemini CLI MCP docs
⚡ Droplet tip: Node.js v18+ required. Run node -v first. If lower, upgrade via nvm install 20. Also use claude mcp list to verify servers loaded correctly in Claude Code.
